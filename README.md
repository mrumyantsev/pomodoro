# Pomodoro

Данное одноэкранное веб-приложение представляет собой реализацию механического кухонного таймера.

По нажатию кнопки "Start" устанавливается таймер на количество минут, указанных в поле для ввода (по умолчанию: 25). По истечении времени таймера воспроизводится звуковой сигнал, похожий на звук механического кухонного таймера. В отличие от настоящего таймера, можно устанавливать запредельное количество минут.

Идея вдохновлена "[методом помидора](https://ru.wikipedia.org/wiki/%D0%9C%D0%B5%D1%82%D0%BE%D0%B4_%D0%BF%D0%BE%D0%BC%D0%B8%D0%B4%D0%BE%D1%80%D0%B0)", для работы по которому создано данное приложение.
