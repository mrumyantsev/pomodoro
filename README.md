# Pomodoro

Данное одноэкранное веб-приложение представляет собой реализацию кухонного таймера.

По нажатию кнопки "Start" устанавливается таймер на количество минут, указанных в поле для ввода (по умолчанию: 25). По истечении времени воспроизводится звуковой сигнал, похожий на звон механического кухонного таймера.

[index.html](./index.html)

## Системные требования

- Windows/Linux/macOS.
- Веб-браузер.

## Как работать по "методу помидора"?

1. Откройте приложение в веб-браузере и держите его первой вкладкой. (Вам все равно понадобится веб-браузер по мере работы.)
2. Запустите таймер на 25 минут, когда решите приняться за работу.
3. Работаете сконцентрированно, не отвлекаясь на посторонние дела, пока не прозвенит звонок.
4. Когда раздастся звонок таймера остановите свою деятельность и дайте себе небольшой перерыв вдали от рабочего места (5-10 минут). (Не рекомендуется продолжать сидеть за компьютером.)
5. Повторяйте шаги 2-4, пока вам нужно выполнять вашу работу. После 4-х "помидор" сделайте большой перерыв (20-30 минут).

## Особенности

- Можно устанавливать неограниченное количество минут.
